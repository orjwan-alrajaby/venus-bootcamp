## Your Task đź§©

You will receive a landing page that uses only `div` and `span` elements.

Your task is to replace them with proper semantic HTML elements wherever appropriate.

These are SOME elements you will probably need:

| Purpose            | Suggested Element       |
| ------------------ | ----------------------- |
| Page intro         | `header`                |
| Navigation         | `nav`                   |
| Main page content  | `main`                  |
| Standalone content | `article`               |
| Page sections      | `section`               |
| Side information   | `aside`                 |
| Quote              | `blockquote`            |
| Images + captions  | `figure` + `figcaption` |
| Contact info       | `address`               |
| Footer             | `footer`                |
| Lists              | `ul` / `ol` / `li`      |
| Important text     | `strong`                |
| Emphasized text    | `em`                    |

You may (or may not) also use/need:

* headings (`h1`, `h2`, `h3`, `h4`, `h5`, `h6`)
* `p`
* `aside`
* `figure`
* `time`
* `ul`
* `ol`
* `li`
* `strong`
* `em`
* `hr`
* and others. This isn't an exhaustive list. Do research to help you decide.

Use the elements that give your content the most meaning.
