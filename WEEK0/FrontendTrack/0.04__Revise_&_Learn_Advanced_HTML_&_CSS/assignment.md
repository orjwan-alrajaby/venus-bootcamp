# Worksheet 1 — Semantic HTML & Meaningful Tags 🏗️

In this worksheet, you will practice using HTML elements that describe the meaning of their content — not just how it looks visually.

You will work with tags such as:

* `header`
* `main`
* `section`
* `article`
* `nav`
* `footer`
* and more

The goal is to build HTML that is easier to:

* read
* maintain
* navigate
* understand by assistive technologies

[GET STARTED.](./0.01__Semantic_HTML/assignment.md)

---

# Worksheet 2 — Accessibility ♿

Not everyone experiences the web using a mouse, keyboard, or screen the same way.

Many users rely on:

* keyboards instead of mice
* screen readers
* assistive technologies
* zoom tools
* high contrast settings

In this worksheet, you will learn accessibility principles such as:

* keyboard navigation
* focus management
* semantic structure
* accessible labels
* screen reader support

Then you will apply those principles in practice.

[GET STARTED.](./0.02__Accessibility/assignment.md)

---

# Worksheet 3 — Forms & Validation 📝

Forms are one of the most important parts of the web.

Without forms, we would not have:

* user accounts
* online shopping
* payment systems
* messaging systems
* online learning platforms

But forms also require structure, validation, and security.

In this worksheet, you will practice:

* building forms
* validating user input
* improving accessibility
* providing meaningful feedback
* handling real-world form scenarios

[GET STARTED.](./0.03__Forms_and_Validation/assignment.md)
