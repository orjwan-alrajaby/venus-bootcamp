# Assignment 3 — Learning Assignment (ES6 & Async JavaScript) 🧠

## Overview

In this week, you will practice:

* ES6 JavaScript syntax
* arrow functions
* array and object methods
* spread/rest operators
* callbacks, promises, and async/await
* debugging and problem solving

This week is different from previous weeks.

You will **not** build a full project.  
Instead, you will solve JavaScript worksheets and focus on core concepts.

This week is about **thinking in JavaScript** 💡

The goal is to understand:

* how data changes
* how functions behave
* how asynchronous code runs
* why code produces certain results

---

## Goal 🎯

By the end of this week, you should:

* write cleaner JavaScript using ES6
* use `map`, `filter`, `reduce`, and `forEach` correctly
* understand when to use array methods instead of loops
* transform arrays and objects
* understand callbacks, promises, and `async/await`
* follow how asynchronous code executes
* read, debug, and explain JavaScript code

## Rules ⚖️

* Use JavaScript only
* No React
* No frameworks
* No backend
* No AI-generated final answers without understanding them
* No copy-pasting solutions from the internet
* Use `const` by default
* Use `let` only when the value changes
* Avoid `var`
* Use arrow functions when requested
* Do not use loops in worksheets that say “no loops”
* Add `console.log()` examples to test your answers
