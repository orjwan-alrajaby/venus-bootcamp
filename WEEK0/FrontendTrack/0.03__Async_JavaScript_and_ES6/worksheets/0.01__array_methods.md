## 🧩 Challenge 2 — map

Convert this array of strings into an array of objects:

```js
const names = ['Ahmad', 'Nour', 'Amira', 'Omar'];
```

👉 Expected output:

```js
[
  { name: 'AHMAD' },
  { name: 'NOUR' },
  { name: 'AMIRA' },
  { name: 'OMAR' }
]
```

👉 Requirements:

* Use `map`
* Convert each name to uppercase

## 🧩 Challenge 3 — filter

Return only countries where language is:

* `"Arabic"` OR `"English"`

```js
const countries = [
  { country: 'Palestine', language: 'Arabic' },
  { country: 'Spain', language: 'Spanish' },
  { country: 'Germany', language: 'German' },
  { country: 'USA', language: 'English' }
];
```

👉 Expected output:

```js
[
  { country: 'Palestine', language: 'Arabic' },
  { country: 'USA', language: 'English' }
]
```

👉 Requirements:

* Use `filter`

## 🧩 Challenge 6 — Spread Operator (`...`)

### a) Merge arrays

```js
const arr1 = [1, 2, 3];
const arr2 = ['a', 'b', 'c'];

// your code here
```

👉 Expected:

```js
[1, 2, 3, 'a', 'b', 'c']
```

---

### b) Add item to array

```js
const fruits = ['Apple', 'Orange'];

// add "Banana" without modifying original array
```

👉 Expected:

```js
['Apple', 'Orange', 'Banana']
```

---

### c) Merge objects

```js
const obj1 = { name: 'Ahmad' };
const obj2 = { age: 25 };

// your code here
```

👉 Expected:

```js
{ name: 'Ahmad', age: 25 }
```

---

### d) Add property to object

```js
const user = { name: 'Ahmad' };

// add "isAdmin: true"
```

👉 Expected:

```js
{ name: 'Ahmad', isAdmin: true }
```
