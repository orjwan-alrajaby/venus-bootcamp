# 🏗️ SECTION: Worksheet 3 — Data Transformation (No Loops)

## 🎯 Goal

Practice:

* transforming data between different shapes
* working with arrays and objects together
* using:

  * `map`
  * `filter`
  * `reduce`
  * `Object.keys`
  * `Object.entries`
  * spread operator

---

## ⚠️ Rules

* Do NOT use loops (`for`, `while`)
* Use array/object methods only
* Do NOT mutate original data
* Use `const` where possible
