# WEEK 1 — Learning Assignment (HTML & CSS)

## Overview

In this week, you will focus on learning:

- HTML
- CSS
- Basic layout (Flexbox)
- Responsive design

You will build the **visual structure** of a movie browsing page based on a Figma design.

---

## Goal

By the end of this week, you should:

- understand how HTML is used to structure a page
- understand how CSS is used to style and position elements
- be able to recreate a UI from a design (to the best of your ability)

---

Figma Design:

## Requirements

### 1. Page Structure

- Create the layout using HTML:
  - movie list section
  - movie details section
  - search input

---

### 2. Styling

- Use CSS to:
  - position elements correctly
  - apply spacing and layout
  - make the UI readable

---

### 3. Responsive Design

Your page should work on:

* **mobile**: 320px to 767px
* **tablet**: 768px to 1023px
* **desktop**: 1024px and above

It does not need to be perfect, but it should:

- not break
- not overflow
- remain usable

---

## What You Are NOT Expected To Do

- No JavaScript
- No interactivity
- No API
